1. 对象方法的补充

1.1 hasOwnProperty

对象是否有某个属于自己的属性(不是原型上的)

1.2 in 操作符

判断某个属性是否在某个对象或者对象的原型上

1.3 instanceof

用于检测构造函数的 prototype 是否出现在某个实例对象的原型链上
就是用于判断对象和(类)构造函数之间的关系

1.4 isPrototypeOf

用于检测某个对象，是否出现在某个实例对象的原型链上
可以判断对象之间的继承关系

2. class 方式定义类

2.1 认识 class 定义类

2.2 class 类中的内容

new 的时候发生了什么事:会调用构造函数的 constructor，并执行如下操作，

  1. 在内存中创建一个新的空对象；
  2. 这个对象内部的__proto__属性会被赋值该构造函数的 prototype 属性；
  3. 构造函数内的 this 会指向创建出来的新对象；执行构造函数体内的代码；
  4. 如果构造函数没有返回非空对象，则返回创建出来的对象。

2.3 类和构造函数的区别

class 定义的类不能作为普通函数去调用,必须和 new 连用
