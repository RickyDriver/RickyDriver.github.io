DOM 操作(一)

1. 什么是 DOM

DOM:Document Object Model 文档对象模型

DOM是在node下的一个对象

DOM 就是 js 和网页中各种内容的一个桥梁，允许开发者去通过 js 操作网页

3.2 DOM 继承关系图

node即代表构成网页的所有元素，其中包括：

  • document，即上面提到的dom
  • element，即html标签里的内容，HTMLelement继承至element，而不是继承至document
  • chracterData,这是一个抽象的接口which contains characters（包含无意义的字符）
  • attr，attribute，指的是标签里的属性值，如id

4 节点、元素的导航

4.1 document 对象

document对象即 整个网页

获取非常规标签
html 元素:document.documentElement
body 元素:document.body
head 元素:document.head
文档声明:document.doctype

4.2 节点(Node)之间的导航(navigator)

节点是指构成网页的最小元素，详见第六节

包括：

  • 注释
  • html标签
  • 文本等等

  • 概念:如果我们获取到一个节点后，可以根据这个节点去获取其他节点，我们称为节点之间的导航
  • 节点之间存在如下关系
    父节点:parentNode
    前兄弟节点:previousSibling
    后兄弟节点:nextSibling
    子节点:childNodes
    第一个子节点:firstChild
    最后一个子节点:lastChild

4.3 元素(Element)之间的导航(navigator)

  • 概念:如果我们获取到一个元素后，可以根据这个元素去获取其他元素，我们称为元素之间的导航
  • 元素之间存在如下关系
    父元素:parentElement
    前兄弟元素:previousElementSibling
    后兄弟元素:nextElementSibling
    子元素:children
    第一个子元素:firstElementChild
    最后一个子元素:lastElementChild

4.4 Document 和 Element 的关系

4.5 表格(table)元素的导航(navigator)

  • table.rows/table.tHead/table.tFoot/table.tBodies
  • table.rows 表格内部的 tr 的集合
  • tr
    tr.cells tr 中所有的单元格的集合
    tr.sectionRowIndex tbody 内的行索引
    tr.rowIndex 整个表格中行的索引，包含了表格头部
  • td.cellIndex 在某一行中某个单元格的索引

4.6 表单(form)元素的导航(navigator)

5 获取元素的方法!!!

5.1 getElementBy*

  1. document.getElementById(‘id 名’)通过 id 获取元素
  2. document.getElementsByClassName(‘类名’)
  3. document.getElementsByTagName(‘标签名’)
  4. document.getElementsByName(‘name 属性值’)

5.2 query 系列

  1. querySelector(‘选择器语法’)
    获取单个

  2. querySelectorAll(‘选择器语法’)
    获取多个，如果只有一个符合，也会以 NodeList 给你

5.3

方法名 搜索方式 可以在元素上调用？ 实时的
querySelector Css-selector ✅ -
querySelectorAll Css-selector ✅ -
getElementById id - -
getElementsByName Name - ✅
getElementsByTagName tag or * ✅ ✅
getElementsByClassName Class ✅ ✅

开发常用的是 query 系列，有时候会使用 getElementById 来兼容低版本浏览器

关于实时的理解

6 Node 节点的属性

  • 什么是节点？
    页面组成的最小单元
  • 注意事项
    标签都是节点

6.1 nodeType(了解)

nodeType 属性提供了一种获取节点类型的方法
他是一个数值类型

  • 8 注释节点
  • 3 文本节点
  • 1 元素节点

6.2 nodeName、tagName(了解)

nodeName 和 tagName 不同？
对于元素，都一样
对于其他节点类型不同，nodeName 有值是对应的字符串，tagName 为 undefined

6.3 data、innerHTML、textContent(掌握)

  • data针对非元素节点获取数据(nodeValue)
  • innerHtml拿到div里面所有的文本包括标签,反之也可以在js对h5添加标签
  • textContent拿到node（div）里的内容，不包括标签

6.4 节点的其他属性

  • DOM 元素的其他属性
    value 常用于表单元素 input/select
    href a 标签
    id 全局属性( id class title style…)
    全局属性是所有元素都可以使用的属性
  • class/style 放到后面讲
  • hidden 也是一个全局属性，可以用于设置元素的显示/隐藏
