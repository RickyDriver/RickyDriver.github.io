for循环每循环一次,里面就会有同名的变量,如果用var定义for循环里的i,每次循环i就会被重置为0,所以我们想通过for循环来对某个以i作为索引的子项做某些操作的时候,就会出现问题.

BOM

Browser Object Model 浏览器对象模型

  • BOM 的顶级对象是 window
    • 所以和 BOM 相关的内容 都是 window.xxxx
    • 但是在书写的时候，可以省略 window 不写

浏览器的窗口尺寸

  • 浏览器的可视窗口的宽度和高度
  • 语法
    1. window.innerWidth 宽度
    2. window.innerHeight 高度
  • 注意:获取到的尺寸是包含滚动条在内的

浏览器的弹出层

  1. window.alert() 提示框
  2. window.confirm() 确认框
    在提示框的基础上多了一个 取消 按钮
    返回值是一个布尔值
    用户点击取消的时候 false
    用户点击确定的时候 true
  3. window.prompt() 输入框
    在确认框的基础上多了一个输入框
    返回值:
    用户点击取消，那就是 null
    用户点击的是确定，那就是输入框的内容

都会中断程序的继续执行，直到用户操作为止

浏览器的地址栏

在 window 对象内有一个属性叫做 location 里面存储的都是和地址相关的东西

  1. href
    是一个可读写的属性
    读:拿到的是当前页面的地址栏的完整信息
    写:修改当前地址栏
  1. reload()

刷新页面
注意：不要写在打开页面就会执行的地方

浏览器的历史记录

window 下有一个成员叫 history，里面存储的都是和历史记录相关的信息

  1. window.history.back()
    回到上一个历史页面
    前提：你必须得有上一个页面，当前的页面必须是从某一个页面跳过来的
    等价于浏览器左上角的 ⬅️ 按钮

  2. window.history.forward()
    去到下一个页面
    前提:你必须得有下一个页面，当前的页面必须是从某一个页面退回来的
    等价于浏览器左上角的 ➡️ 按钮

  3. window.history.go()
    正整数 前进
    0 重新打开当前页面
    负整数 回退

浏览器的标签页

  1. open()
    window.open(“地址”)
    开启一个新的标签页
  2. close()
    window.close()
    关闭当前标签页
    只能关闭用 open 打开的页面

浏览器常见事件

  1. onload
    window.onload = function(){}
    页面所有的资源(html 结构，音频，图片…)加载完毕后触发
    当你把 js 写在 head 里面，而且还要操作元素的时候可以用
  1. onresize
    页面可视窗口大小发生变化的时候触发
    响应式
    移动端横屏判断
  1. onscroll
    页面滚动条改变位置时触发
    window.onscroll = function(){}

浏览器卷去的尺寸

分为卷去的高度 和 宽度
document.documentElement.scrollTop 有文档声明才能用 卷去的高度
document.documentElement.scrollLeft 有文档声明才能用 卷去的宽度
document.body.scrollTop 没有文档声明才能用 卷去的高度
document.body.scrollLeft 没有文档声明才能用 卷去的宽度

浏览器滚动到

  1. scrollTo()
    window.scrollTo(x,y)
    x,y 都是数字
    x:设置卷去的宽度
    y:设置卷去的高度
    必须传递两个参数
    可以平滑的滚动，需要在样式里设置html{scroll-behavior:soomth;}

传递一个对象类型
window.scrollTo({
top:yyy,
left:xxx,
behavior:’smooth’
})
可以只写一个值，也可以平滑滚动
