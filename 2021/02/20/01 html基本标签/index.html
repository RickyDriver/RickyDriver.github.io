前端开发学习笔记 02 html基本标签

发表于 2021-02-20 更新于 2023-03-26 分类于 笔记

html 基本标签

标题标签

总共有 6 级标题

段落标签

段落标签是一个块级标签，前后添加了换行符

文本标签 span

文本标签是行内标签，他不会独占一行，而且会和其他的行内标签共行

换行标签 br

在需要换行的地方添加 br 标签即可换行
换行标签是一个单标签

  • 双标签:由一个开始标签和一个结束标签组成 比如<p></p>
  • 单标签:只有一个 <br/>

分割线 hr

分割上下文

图片标签

在网页中插入图片用

常用属性

  • src 图片的地址
  • alt 图片加载失败显示的文字
  • width 图片的宽度
  • height 图片的高度
  • title 鼠标悬停时，显示的文字内容
  • 相对地址:相对于文件所在的位置进行引入资源的路径，对当前文件所在位置非常敏感
  • 绝对地址:不管文件在哪里，地址不变，图片始终能够正常引用

路径特殊符号

  • ./ 代表当前目录
  • ../ 上一层目录
  • ../../ 上上一层目录

容器标签 div

主要作用是组合其他 html 元素，他在语意上不代表任何特定的类型，但是他可以将某些内容按照需求来分组.

超链接标签 a

<a href="http://www.baidu.com" target="_blank">点我去百度</a>
常用属性

  • href 超链接指向的地址，也可以是某个锚点
  • target 超链接的打开方式
    • _self 默认值(缺省值)当前页面打开
    • _blank 在新标签页中打开

a 标签的应用 锚点跳转

有两种方式可以实现返回页面最上部，使用name或者id属性

  • 利用元素的 id

第一步:在需要跳转的标签中插入 id
第二步:在 a 中 href 填上对应的 id 即可 不要忘记加#

  • 利用 a 标签的 name 属性

文本格式化标签

字符实体

在 html 中，有一些字符是已经预留好的，由于 html 标签是由<>包裹起来的，所以当内容存在这些符号时,浏览器可能误认为他们是标签。为了正确的显示这些字符，就采用了实体字符

字体字符

列表

列表分为 有序列表，无序列表，自定义列表

有序列表

有序列表的属性

  • type 指定标号的类型
    • 1 数字(默认)
    • a 小写字母编号
    • A 大写字母编号
    • i 小写罗马数字编号
    • I 大写罗马数字编号
  • start 指定列表的起始编号
  • reversed 列表是否倒叙，写上就是倒叙

无序列表

type 用于设置列表头部的样式

  • circle 空心圆
  • square 实心方块
  • disc 实心圆
