前端开发学习笔记 16 flex布局

发表于 2021-03-20 更新于 2023-03-25 分类于 笔记
本文字数： 1.2k 阅读时长 ≈ 4 分钟

flex

Flexbox 翻译为弹性盒子

  • 弹性盒子是一种用于按行或者按列布局元素的一维的布局方法;
  • 元素可以膨胀以填充额外的空间，收缩以适应更小的空间
  • 通常我们用 flexbox 来布局的方案称之为 flex 布局(flex layout)

flex 布局的重要概念

两个重要的概念

  • 开启了 flex 布局的元素叫 flex container
  • flex container 里面的直接子元素叫做 flex item

当 flex container 中的子元素变成 flex item 时，具备以下一些特点

  • flex item 的布局将受到 flex container 的属性来控制
  • flex item 不再严格区分块级/行内级元素
  • flex item 默认情况下是包裹内容的，但是可以设置宽高

设置 display 属性为 flex 或者 inline-flex 可以成为 flex container

flex 的属性

  • flex container 上的属性
    1. flex-flow
    2. flex-direction
    3. flex-wrap
    4. justify-content
    5. align-items
    6. align-content
  • flex item 上的属性
    1. flex-grow
    2. flex-shrink
    3. flex-basis
    4. order
    5. align-self
    6. flex

flex-direction 主轴方向

flex items 默认都是沿着主轴方向从 main start 开始往 main end 排布

  • row(默认值)
  • row-reverse 行反转
  • column 列排布
  • column-reverse 列反转

flex-wrap

flex-wrap 决定了 flex container 是一行还是多行

  • nowrap 默认 单行
  • wrap 多行
  • wrap-reverse 多行而且 cross start 和 cross end 反转

flex-flow

flex-flow 是 flex-direction 和 flex-wrap 的简写

  • 任何顺序，而且两个可以随便省略

justify-content

justify-content 决定了 flex items 在 main axis 上的排列方式

  • flex-start 默认值 与 main start 对齐
  • flex-end 与 main end 对齐
  • center 居中对齐
  • space-between
    • flex items 之间距离相等
    • 与 main start ，main end 两端对齐
  • space-evenly
    • flex items 之间距离相等
    • flex item 与 main start，main end 之间的距离 等于 flex item 之间的距离
  • space-around
    • flex items 之间距离相等
    • flex item 与 main start，main end 之间的距离 等于 flex item 之间的距离的一半

align-items

align-items 用于控制交叉轴上的排布方式

  • normal 在弹性布局中和 stretch 一样
  • stretch 当 flex item 在 cross axis 方向上的 size 为 auto 时 会自动拉伸填充 flex container
  • flex-start(默认)：与 cross start 对齐
  • flex-end：与 cross end 对齐
  • center 居中对齐
  • baseline 基线对齐

align-content

align-content 决定了多行的 flex items 在 cross axis 上的对齐方式，用法和 justify-content 类似

  • stretch（默认）：与 align-items 的很相似
  • flex-start：与 cross start 对齐
  • flex-end：与 cross end 对齐
  • center 居中对齐
  • space-between
  • space-around
  • space-evenly

order

order 决定了元素的顺序

  • 默认是 0
  • 可以设置任意整数，值越小排在越前面

align-self

flex items 可以通过 align-self 属性来覆盖 flex container 容器设置的 align-items

  • auto(默认):遵循容器设置的值
  • 其余的值和 align-items 一样

flex-grow

flex-grow 决定了 flex items 是如何拉伸的

  • 可以设置任意的非负的数字(正整数，正小数，0) 默认是 0 不成长
  • 当 flex container 在 main axis 上有剩余的 size 时，flex-grow 才能生效，如果所有 flex item 的 flex-grow 属性总和 sum 超过了 1，那么每个 flex items 扩展的 size 为

flex container的剩余size*flex-grow/sum

flex-shrink

flex-shrink 决定了 flex items 如何缩小

  • 可以设置任意的非负的数字(正整数，正小数，0) 默认是 1 可以被压缩
  • 当 flex items 在 main axis 上超过了 flex container 的 size flex-shrink 才会生效

如果所有的 flex items 的总和超过 1，每个 flex items 它收缩的 size 为

flex items超出的container的size*收缩比例/所有flex items的flex-shrink的之和

flex items 收缩后 最终展现出来的宽度不能小于 min-width

flex-basis

用来设置 flex items 在 main axis 方向上的 base size

  • auto(默认值) 、具体的宽度数值

决定 flex items 最终的宽度的影响因素有多个,优先级从高到低

  • max-width\min-width\max-height\min-height
  • flex-basis
  • width/height
  • 内容本身的 size

flex

是 flex-grow、flex-shrink、flex-basis 的缩写属性

flex:1
grow 为 1 shrink 为 0 basis 默认
