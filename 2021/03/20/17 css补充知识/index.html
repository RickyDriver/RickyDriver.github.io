URL 地址

认识 url

  • url 统一资源定位符(Uniform Resource Locator)
  • url 无非就是给定的独特资源在 web 上的地址
    • 每个有效的 url 都指向一个唯一的资源
    • 这个资源可以是一个 html 页面，一个 css 文档，一副图像…

URL 的格式

  • url 的标准格式如下:
    • [协议类型]://[服务器地址]:[端口号]/[文件路径][文件名]?[查询]#[片段ID]
    • https://john.deo@www.example.com:123/forum/question/?tag=network&news#top

URI 的区别

  • URI:Uniform Resource Identifier 统一资源标识符，用于标识web 技术使用的逻辑或者物理资源
  • URI 在某一个规则下能把一个资源独一无二的识别出来
  • URL 是 URI 的一个子集

Locators are also identifiers,so every URL is also a URI,but there are URIs which are not URLs.

全局属性data-*

  • 在 html5 中，新增的一种全局属性的格式data-*,用于自定义属性
    • data 设置的属性可以在 js 中的 dom 操作通过dataset拿到
    • 通常用于 html 和 js 数据之间的传递
  • 在小程序中，就是通过data-*来传递数据的，所以必须掌握

css 常见的函数扩展

  • 在前面我们已经使用过了很多个 css 的函数
    • 比如 rgb/rgba/translate/rotate/scale/…
    • css 函数通常可以帮助我们更加灵活的来编写样式的值
  • var/calc/blur

css 函数 -var

  • css 中可以自定义属性
    • 属性名需要以两个减号(–)开始
    • 属性值可以是任何有效的 css 值
  • 我们可以通过 var 函数来使用
  • 推荐将自定义属性定义在 html 中，往往使用:root选择器

css 函数 - calc

  • calc（）函数允许在声明 css 值的时候执行一些计算
    • 计算支持加减乘除
    • + 和 - 运算符两边必须有空白字符
    • 通常用来设置一些元素的尺寸或者位置

css 函数 - blur

  • blur()函数将高斯模糊应用于输出图片或者元素
    • blur(radius)
    • radius，模糊半径，值越大越模糊
  • 通常会和其他属性一起使用
    • filter:应用于元素
    • backdrop-filter:为元素的后面添加模糊/其他效果
