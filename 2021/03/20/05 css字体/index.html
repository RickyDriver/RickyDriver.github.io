字体

字体类的样式

使用 font 属性必须按照上述顺序来书写，而且font-size和font-family是不可忽略的。
每个参数仅允许有一个值，忽略将会使用默认值。

实际开发，更倾向于分开来设置字体属性

字体风格 font-style

设置元素的字体风格(是否倾斜)

  • 默认 normal 指定字体的文本样式为正常的字体
  • italic 指定字体的文本样式为斜体 展示该字体的斜体
  • oblique 斜体 强行把字体斜着
  • inherit 继承 继承父元素的字体样式

字体粗细 font-weight

  • 默认值:normal 正常字体 相当于数值 400
  • bold:粗体，相当于数值 700
  • bolder:定义比继承更重的字体
  • lighter:定义比继承更轻的字体
  • integer:用数字来表示字体的粗细，100-900
  • bolder 和 lighter 表示的字体的粗细是基于从父元素继承而来的字重计算得到的，与 normal 和 bold 所表示的字重无关

字体大小 font-size

  • length 和% 是平时开发中最常用的书写方式
  • em 主要用于移动端，是相对于父元素字体大小的一个相对长度
  • 网页中最小的字体:12px
  • chrome 默认字体大小:16px

使用方式

文本

line-height行高

设置当前元素中每行文本的高度

normal:默认，合理的行高
length:设置固定的，例如 30px 的行高
%:用百分比指定行高，基于文本的font-size进行计算

使用 line-height 时尽量不要让行高小于字体大小，这样会导致多行文本的情况下，发生上下行文字重叠的情况

单行文本居中:设置行高为容器高度

文本装饰text-decoration

设置元素的文本修饰，比如下划线，上划线，删除线。

  1. text-decoration-line
    none 没有
    underline 下划线
    overline 上划线
    line-through 贯穿线
  2. text-decoration-style
    solid 单实线
    double 双实线
    dotted 点划线
    dashed 虚线
    wavy 波浪线
  3. text-decoration-color
    设置装饰线的颜色

实际开发中我们往往只需要用到第一个
text-decoration-line

首行缩进 text-indent

text-indent: 2em;

为什么text-indent最常用的单位是 em
假设当前的段落字体大小 16px，如果你希望一个段落的文字能够缩进两个字体大小，可以用text-indent:32px;;也可以使用text-indent:2em。假如后期段落字体大小调整为 14px，如果之前写的是 32，现在就需要修改为 28px，但是如果是 em 的话，就不需要任何变动.

水平文本对齐text-align (重点)

设置元素中文本的水平对齐方式
left 左对齐
right 右对齐
center 居中对齐
start 内容对齐开始边界
end 内容对齐结束边界
justify 内容两端对齐 但是对于强制打断的行以及最后一行(包括 only 一行文字的情况)不会两端对齐而是遵循默认的对齐方式

word-spacing 和 letter-spacing

word-spacing 单词间隔
letter-spacing 字母间隔

简单来说，letter-spacing添加的是字母(中文文字)之间的空白;word-spacing添加每个单词之间的空白，对中文无效
