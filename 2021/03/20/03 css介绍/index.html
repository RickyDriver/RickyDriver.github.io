前端开发学习笔记 04 css介绍

发表于 2021-03-20 更新于 2023-03-26 分类于 笔记
本文字数： 1.5k 阅读时长 ≈ 5 分钟

css

层叠样式表
美化网页

语言特点

从上往下执行，同名覆盖
三大特性: 继承性，优先级，层叠性

  • 结构与表现分离(html 只负责结构,使得页面更加的简洁)
  • 减少开发的工作量(子元素可以继承父元素的样式，减少了重复的代码)
  • 提高了页面的浏览速度
  • 易于升级

厂商前缀

浏览器厂商通过厂商前缀来标记试验性的功能或者属于该厂商的浏览器专属属性
Chrome -webkit-
Opera -o-
IE -ms-
FireFox -moz-

引入方式

  1. 行内(内联)样式
    <header style="font-size: 12px"></header>
  2. 页内
    直接在 html 的头部中style进行设置
  3. 页外
    在头部 通过 <link rel="stylesheet" href="">引入

选择器

1. 元素选择器

通过一个简单的元素选择器，就能大范围的将所有该元素的样式进行统一的修改

2.群组选择器

3.通配符选择器

通配符选择器,能匹配到所有元素

页面中所有的字体颜色都会变成红色

4. 类 和 id 选择器

类选择器
通过 class 属性添加，css 的样式是通过.+类名去选择到对应类名的标签

id 选择器
通过 id 属性添加，css 的样式是通过#+id 名来选择到对应的标签

5.权重问题

!important 无限大权重
行内样式 优先级 1000
id 选择器 0100
类选择器 0010
元素选择器 0001
通配符选择器 0000

6.关系选择器

关系选择器，是根据文档的结构来进行选择。

同时选择两个元素

子代选择器

使用>分隔,来选择某个子代的元素

相邻选择器

使用+来分隔，选择与该元素相邻的元素

此时只有内容为‘经过多年的摸爬滚打’的 span 标签字体颜色有变化，其他的 span 标签字体颜色则没有变化

兄弟选择器

伪类选择器
