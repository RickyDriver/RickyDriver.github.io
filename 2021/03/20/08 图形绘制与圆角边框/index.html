图形绘制与圆角边框

图形绘制

理解三角形如何绘制

此时我们会得到一个花花绿绿的图形
既然每个边框都能设置颜色，能不能设置成透明的

得到了一个等腰梯形,而且发现这个等腰梯形窄边的宽度由内容宽度来决定，当我们把盒子的宽度设置为 0 的时候，就形成了一个尖角，这样就得到了我们的三角形

三角形的宽高与 width 和 height 无关，是由边框的厚度来决定的

小练习

圆角边框

小米 200w 图标
画圆，设置border-radius为宽高的一半或者 50%都可

圆角边框
border-radius:lenght|%

  • 一个值:4 个角的值相同
  • 两个值:左上和右下；右上和左下
  • 三个值:左上；右上和左下；右下
  • 四个值:左上；右上；右下；左下

盒子阴影

x 轴偏移量:正值右移动，负值左移动
y 轴：正值上移动，负值下移动
模糊半径：值越大越模糊 为 0 时 和原来的图形一样
阴影扩散范围：正值放大，负值缩小
投影方式：inset/outset 内阴影/外阴影

实际开发我们调阴影会选择阴影来加快效率
