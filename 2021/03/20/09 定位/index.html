前端开发学习笔记 10 定位

定位

常见的布局方式：浮动，定位，弹性，(网格)。

多个层叠状态，会优先考虑定位布局。如：弹窗，返回顶部按钮…

语法

元素可以使用顶部，底部，左侧，右侧这四个定位属性；然而这四个属性无法工作，除非你设定了 position 属性。

static 静态定位

html 元素默认值，没有定位，遵循正常文档流的现象，静态定位的元素不会收到 top,bottom,left,right 的影响。

relative 相对定位

relative 表示，相对默认位置(static)进行你要的偏移，即相对定位的基准点是元素的默认位置(static)，遵循正常文档流

移动相对定位的元素，但是它原本所占的空间并不会改变
相对定位的元素，常常被用来作为绝对定位元素的容器

absolute 绝对定位

absolute 表示，相对于最近的一个定位属性不为static的祖先元素，如果都没有，则一直回溯到 html 元素。脱离文档流

fixed 固定定位

元素的位置相对于浏览器窗口固定
即使是窗口滚动它也不会动

fixed 使得元素脱离文档流，也不占据空间
fixed 元素可以和其他元素重叠

三个特点

absolute 和 fixed 的元素有三个特点
块状化，包裹性，破坏性。

粘性定位

在设置的值(阈值)到达前都是 relative，到达后变成了 fixed
阈值，一定要设置;top``bottom``left``right 必须有一个，才能使得粘性定位生效

sticky由于使用时bug多，不推荐使用。
